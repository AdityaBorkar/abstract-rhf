# Getting Started

## Installation

```bash
bun add formzen
```

## Basic Usage

Here's a simple example of how to create a form:

```tsx
// Filename: page.tsx
import { useForm } from "formzen";

export default function MyForm() {
  const { Form } = useForm({
    defaultValues: {
      username: "",
      email: "",
    },
    onSubmit: (data) => {
      console.log(data);
    },
  });

  return (
    <Form>
      <input {...form.register("username")} />
      <input {...form.register("email")} />
      <button type="submit">Submit</button>
    </Form>
  );
}
```
