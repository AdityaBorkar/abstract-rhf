# Background

{/* https://tanstack.com/form/latest/docs/philosophy */}
{/* https://stylexjs.com/docs/learn/thinking-in-stylex/ */}

We believe that Schema Libraries is the way - how much small or big the form be! Schema libraries make it simpler to handle form data in local as well as server environments.


## Bring your own Form Library

- Tanstack Forms
- React Hook Form

(there's a catch - you can use any one of the two - we have no plans to support any other library.)

## Bring your own Schema Library

- Yup
- Zod
- Joi
- Ajv
- Effect
- ArkType
- Valibot
- Superstruct
- Standard Schema

## Features

- Designed for local-first apps
- TypeScript first
