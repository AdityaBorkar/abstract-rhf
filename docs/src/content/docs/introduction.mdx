# Introduction

Welcome to @forms - a modern form library for React applications that combines the power of React Hook Form with the beauty of ShadcnUI.

## What is @forms?

@forms is designed to make form handling in React applications simpler and more intuitive while maintaining full type safety and a great developer experience.
